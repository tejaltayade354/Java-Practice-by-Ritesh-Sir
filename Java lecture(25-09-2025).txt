CONSTRUCTORS:
- Constuctors can be defined as special methods, which are meant to perform operation but do not have a return type.
-rules to create a constructor:
constructor name should be same as class name.
they can accept arguments.
But they do not have a return type.

-constructors are always called during object creation.(example 1 point.)

EXAMPLE 8
IQ:  What is the use of contructors in real time?
---> in real times, the constructors are primarily used to perform 2 imp operations: 1) To initialize the non static members IN CONSTRUCTOR.
2)To implement the business logics during object creation.

class example8
{
    int a;  //  initially without declaring 0----> then we made 10 in constructor 10
    example8(){
        System.out.println("inside demo constructor " +a);
       a=10;   // non static member initialization
    }
    void m1(){
        System.out.println("inside m1 method " +a);
        
    }
    
    public static void main(String[] args){
        example8 ref=new example8();
        ref.m1();
        System.out.println();
    }
}


inside demo constructor 0
inside m1 method 10
