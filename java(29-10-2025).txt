-startsWith()
-this method checks for the prefix sequence within the starting of the string if it matches, it will return true else it will return false.


-endsWith()
-it will check if the suffix of the string is matching or not at the end of the string, and based on that it will return a Boolean value.

-contains()
it will check whether a character sequence is present anywhere within the string, based on which it will either return true or will return false.


-isEmpty()
-this method will be responsible to check, if a string is empty or not based on which it will return a Boolean value.

SUBSTRING:
- A substring can be defined as extracting a part of the string is called as substring.

substring(int beginIndex)
-this method, it will start extracting the string from the begin index till the end of the string.

substring(int beginIndex, int endIndex)
-in the substring, the begin index and the end index, it will start extracting from the begin index and will extract till the end index -1.

toUpperIndex()
-This will be responsible to convert lower case string to the uppercase.after that return the respective string.

toLowerIndex()
-This will be responsible to convert upperer case string to the lowercase.after that return the respective string.

trim()
-the trim method will be responsible to trim the spaces on either side of the string, and after trimming it will return the string.

-note: trim method will not trim the spaces in betn the string.
