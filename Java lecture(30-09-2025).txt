Example1:

-static blocks are executed during class loading , there is no need to create an object to call static block.
-static blocks are executed before main when they are part of the same class.

class example1 {
    static{
        System.out.println("inside static block");           //static blocks are executed before main
    }
    public static  void main(String[] args){

        System.out.println("start");
        System.out.println("endddddddddddddd");
        // no need of object creation for executing static block, they are executed during class loading.
    }
}

Example2:
-we can create min 1 static block or we can create n no of static blocks.
-If there are multiple static block, then the order of execution will be from top to bottom.

class example2 {
    static{
        System.out.println("inside static block-1");           //static blocks are executed before main
    }
    static{
        System.out.println("inside static block-2");           //static blocks are executed before main  ////// from top to bottom order
    }
    public static  void main(String[] args){

        System.out.println("start");

    }
}

q1) What are main pillars of oop?
--> there are mainly  4 pillars:
     1)INHERITANCE, 2)POLYMORPHISM, 3)ABSTRACTION, 4)ENCAPSULATION.

1) INHERITANCE:
-The class which acquires the members of another class, this process is called inheritance.
-To achieve inheritance there should be is-a relationship between classes.
- To implement inheritance , min there should be 2 classes.
-To represent is-a relationship in java, we use extends keyword.
-BENEFITS of inheritance:
reusability:
duplication less- duplicate code used is none.
length less- length of the code is reduced.
memory less- memory usage is less/optimized.
execution time is also less,
speed of application is faster.
