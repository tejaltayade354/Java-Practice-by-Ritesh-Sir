write a java program to print numbers from 1 to 5 using the do-while loop.


class  dowhile1
{
	public static void main(String[] args) 
	{
		int i=1;
		do
		{System.out.println("DO-while"  +i);
		i++;
		}
		while (i<=5);
		
	}
}

continue
- this statement acts like skipping statement.
- it is responsible for skipping the statements and move the java control at the end of the for loop.( not outside the for loop)
class  continue1
{
	public static void main(String[] args) 
	{
		for(int i=1;i<=5; i++)
		{
			System.out.println(i);
			if(i==3)
			System.out.println("The value of i" +i);
			continue;  // skip
		}
		System.out.println(i);
	}
}
