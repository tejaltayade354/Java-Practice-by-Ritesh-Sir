Boxing and Unboxing Operation:

-Directly converting a primitive data in the form of wrapper objects is called as Boxing Operation.
-Directly converting a wrapper object to primitive type is called as Unboxing Operation. 

imp note:
Boxing and Unboxing feature is by default used in collections, which makes the collections more stronger.

-Example 8:

class Example9 {
    public static void main(String[] args){

    //boxing
    Integer i1=98;
    System.out.println(i1);
    System.out.println(i1.toString());
    //unboxing
    float f1=new Float(67.5f);        //- compiler rewrites as (new Float(67.5f)).floatValue();
-

-Inner classes are categorized mainly into 2 types:
1)Static inner classes
2)Nonstatic inner classes.
-there are 3 types of classes in non static inner classes:
1)normal inner classes.
2)method local
3) Anonymous

1)normal inner classes:
-whenever we compile a java prog, regardless, whether it is a normal class, or inner class, it will generate a .class(dot class) file.
-whenever a inner class .class file is generated, it will contain a $(dollar ) symbol, which will involve outer class name and inner class name.(Outer$Inner.class)
