Custom Exceptions:

exceptions can be categorized mainly into 2 types:

- predefined exceptions
  user defined exception(customized exceptions)

1)predefined exceptions

- they are created by oracle or java.


2) user defined exception(customized exceptions)
- created by users.
-user defined exceptions are categorized into 2 types:
1) custom checked 
2) custom unchecked

-1) custom checked:
- they are also catehgorized into 2 types, 0 arg constructor, arg constructer.

-1) custom unchecked:
- they are also categorized into 2 types, 0 arg constructor, arg constructer.


- if we have to create a custom exception, then we have to extends Exception. class(checked)
