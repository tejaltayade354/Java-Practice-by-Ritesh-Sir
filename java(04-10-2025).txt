instance blocks in inheritance:
-under the concept of inheritance, whenever the child class constructor is called, it will first give the preference to the parent class constructor.
-when the control goes to the parent class constructor, it will check if instance block in parent class is present or not, if present it will execute and only then execute the parent constructor.
-After that the control comes to the child constructor, the child constructor also checks whether instance block is present or not, if present it will execute  and only then it will execute the child class constructor.


-even instance blocks are not part of inheritance.
