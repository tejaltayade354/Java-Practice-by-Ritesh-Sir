SWITCH STATEMENT:

Program:

 class switch1
   {
	public static void main(String[] args) 
	{
		System.out.println("Start @ cyber success");
        int a=2;
		switch(a)
		{
			case 1:
				System.out.println("inside case1");
			break;
			case 2:
				System.out.println("inside case2");
			break;
			default:
				System.out.println("indefault");
			break;
			
		}
               System.out.println("Stop @CyberSuccess.");
	}
   }

-In a switch statement, there is no condition matching.
-here, the way it works is, the argument within the switch is matched or mapped with the  case lable, if the matching happens then the respected statements within the case get executed, and after which due to the break statement the control will be broken, which, which means control is outside the switch statement.
-if the switch argument does not match any of the cases, then by default it will match the default case.

NOTE:

we can use default and cases anywhere within switch statement.


class switch8
{
 	public static void main(String[] args)
