LOOPING STATEMENT:
-A loop can be defined as repeating a statement or instruction no of times, itself is called as a loop or iteration.
-There are three looping statement or iteration statements supported by java.
 1)for (again 3 types -for i, nested for, for each )
 2)while
 3)do-while

1) For loop 
- For loop categorized into 3 types: 1) for i    2) nested for     3)for each(will discussed in arrays and collections)

-SYNTAX:

    for(declaration/initialization; condition; increment/decrement)
        {
          //statements to be executed;
        }


-INCREMENT AND DECREMENT OPERATORS:
operaters  can be  categorized into 2 types-1) increment operator
                                            2) decrement operator
- increment and decrement operators can be categorized into 2 types- 1) pre-increment    2)post increment           
                                                                   - 1) pre-decrement    2) post decrement        
  
1) pre-increment  ++i  and post increment i++ :         
- The pre and post increment operators will increment the value of i or any other variable by 1.

prog:

1) pre-increment AND post-increment          


class   IncPrePost1
{
	public static void main(String[] args) 
	{
		int i=1;
		System.out.println(i);//1
		//preincrement - first increment i value by 1 and then print value of i
		System.out.println(++i);// increments, gives 2  then prints 1
		//post increment- first print the i and then increment the value of i
		System.out.println(i++);// first prints i means 2 then increment the value , we get 3
		System.out.println(i);//3
        }
}
