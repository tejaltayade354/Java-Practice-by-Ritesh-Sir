-example 7:

- to access the inner class object, we use this keyword.
-but to access, outer class instance variable when both the variables names are same, in such a situation, we go for outer.this



class Outer{
    int a=3;
class Inner {
    int a=33;
    void m1(int a){               //all variables have same name
        
        System.out.println("Inside m1 method");
      
        System.out.println(a);     // ->  333
        
        System.out.println(this.a);     // -> 33
        
        System.out.println(Outer.this.a);    // ->   3 


    }
      
    }
}

class Example7 {
    public static void main(String[] args){
    new Outer().new Inner().m1(333);
    }
}

33

-example 10:


-it is possible to extend the inner class to the outer class using keyword extends.


class Outer{
    int a=3;
class Inner extends Outer {
    int a=33;
    void m1(int a){               //all variables have same name
        
        System.out.println("Inside m1 method");
      
        System.out.println(a);     // ->  333
        
        System.out.println(this.a);     // -> 33
        
        System.out.println(super.a);    // ->   3        //inheritance is supported.


    }
      
    }
}
class Example10 {
    public static void main(String[] args){
    new Outer().new Inner().m1(333);
    }
}

2) Method Local Inner Class

-A class declared or created within a method, such a class is called as Method local inner class.

-when the method is called within the stack , only then the following steps can be implemented.
-1) the class can be declared and the object can be created.
2)With the help of object, we can make changes to the respective class members.
3)Once the method leaves the stack, the object will also eligible for disruption.(gets destroyed)


-example 11:


-Whenever we compile, a method local inner class,  it will generate the .class file.
And in that  .class file,  a numeric 1 will be added after $ and before inner(Outer$1Inner.class )

class Outer{
 void m1(){
    System.out.println("inside m1 of outer");
    
    class Inner{
           boolean status=true;
    }
    Inner i=new Inner();
    System.out.println(i.status);
 }

}class Example11 {
    public static void main(String[] args) {
    new Outer().m1();
}}
