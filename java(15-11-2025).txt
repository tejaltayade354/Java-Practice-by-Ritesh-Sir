-A variable at any point of time can store only values but to store multiple value at a time, we go for an array.
-As every variable has a name, every array also has a name.
-Whenever we store any values in an array, its is always store in an index order.
-there are 2 ways to create an array:
1)In an first approach of an array there are 3 stages: 1. declaration    2.Instantion(object creation)       3. initialization

Type[] Name =new type[size];           //         Type[] Name ---> declaration               Name =new type[size];---------->instantiation

ex1:  int[] a=new int[3];

       a[0]=10;               //this 3 lines are initialization
       a[1]=20;
       a[2]=30;

-ARRAY  by nature are homogenous.(one time store value).

IQ: DIFFER BETWEEEN LENGTH() METHOD AND LENGTH VARIABLE?

length()-string->num of letters within string
length -variable->array->size of the array.

-length() method is used to calculate the length of the string which will return no of letters or number or special characters with in a string.
-but the length variable is used to calculate the size of the array.

-example 7:

-Control flow statement:

looping-> for

for each loop:
int [] a ={10,20,30};

for(store the data based on extracted data : from where to extract data/info )

for(int aa : a){
  System.out.println(aa);
}


-The way for each works is will by default start index 0, and extract the data and store it.
-While storing the type should be same as extracted data.
-for each loop will keep an extracted data till the end of the array by using auto increment feature.
-The for each loop will identify the end of the array with out using any condition.

 class Ex7 {
