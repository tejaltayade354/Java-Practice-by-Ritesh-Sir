example 24:

-INSIDE JAVA 8, it is allowed to create a complete methods in interfaces.
-here comes the java 8 default keyword, to inform the JVM  and compiler that we can create a complete method in interface.


-whenever we prefix the method by default keyword within an interface, it is still by nature is considered as public.
-the default keyword can only be used in interface, that is part of java 8 features.

interface it1
{
    default void m1(){         //  in java 8, complete methods are allowed if  they are prefixed by default keyword.
        System.out.println("inside m1 method");
    }
}
class example24 implements it1{
    public static void main(String[] args) {
    example24 ref=new example24();
    ref.m1();
}
}

    void productcategory();

    void searchproducts();

}
class amazonadapterclass implements amazon{                   //adapter class
    public void productcategory(){}                                 
    public void addtocart(){}
    public void searchproducts(){}
}

   public void productcategory(){
        System.out.println("logics for product category");
    }
}

        amazonprodcat cust=new amazonprodcat();
