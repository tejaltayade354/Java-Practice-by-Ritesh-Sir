BREAK:
- irrespective of the condition is true or false, break statement will exit the loop.
-whenever the break statement is encountered, control is sent outside the loop.
 

class b1 {
    public static void main(String[] args) {
         System.out.println("loop start" );
        for(int i=5;i<=10;i++){
            
            System.out.println(i);
            if(i==8){
                System.out.println("end");
                break;
                
            }
        }
    }
}

continue:

- control is moved to the next iteration  skipping everything below continue statement in loop for that iteration.
