Exceptions:

-An exception can be defined as an unexpected event, which may occur during compile time or run time.

-whenever an exception occurs, it will lead to 2 things :

1) it will lead to abnormal termination.
2) the remaining statements will not get executed.

-   What are the different types of exceptions?
--> There are mainly 3 types of exception, 1) UNCHECKED EXCEPTION,  2)CHECKED EXCEPTION,  3)ERROR 


1)UNCHECKED EXCEPTION:
- Whenever the compiler do not check the exception, but JVM checks such an exception is called as unchecked exception.

example 1:

-in java the expression 10/0 will lead to arithmetic exception.

class example1{
    public static void main(String[] args){
        System.out.println("start ");         //print statements before exception
        System.out.println(10/0);
         System.out.println("enddddddddddddddd ");
    }
}

op:

start

        at example1.main(example1.java:4)
