overriding:
-to achieve overriding there are 2 conditions is to be made:
1) minimum it should have two classes.
2) Between the 2 classes there should exist a IS-A relationship.(inheritance)
- The process of inheriting the member from parent to child then changing its behaviour , such a process is called as OVERIDDING.

-RULES OF OVERIDING:

NOTE:
- A method signature can be defined as combination of method name and argument. 
  methodname(argument)

RULE1:
-To achieve overriding, the parent method signature and child method signature should be same.
-example14:

class parent{
    void m1(){
        System.out.println("inside m1 method-parent");
    }

} 
class child extends parent{
    void m1(){     // rule1: method signature should be same
        System.out.println("inside m1 method-child");
    }

} 
 
 class example14 {
    public static void main(String[] args) {
        child c=new child();
        c.m1();   //object is of child ---->mg to tyachya class chya method la prefer karel
    }
}

-RULE5:

TYPECASTING :
-It is a process of converting 1 form to another.
-Tpyecasting can be categorized in one of the approaches- as upcasting and downcasting.
-UPCASTING is a process when a child object will behave as parent.In this process, the child object will loose its characteristics and will acquire the characcteristics of the parent only.
-DOWNCASTING is a process, when a child behaving as a parent, will behave back as child.
-if the child object which is downcasted can acquire not only child class members but can also acquire parent class members.
-IMP NOTE:
-only upcasted objects are downcasted.
-imp note:  direct downcasting is not possible.





-example21:

class parent{
    int a=10;

}
class child extends parent{
    float b=10.1f;

}
class example21 {
    public static void main(String[] args) {
        //upcasting
        parent p=new child();
        System.out.println(p.a);
        // System.out.println(p.b);    //ce
         System.out.println("-----------------------------------");
         //downcasting
          child c= (child)p;
           System.out.println(c.a);
            System.out.println(c.b);
    }
}
