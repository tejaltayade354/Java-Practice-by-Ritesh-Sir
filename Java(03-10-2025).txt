class C extends A{

    }

class D extends C     // single inheritance
{
}
class E extends B         // single inheritance 
{
}

SUPER keyword:
-it is used to store the parent class or the super class object address.
-super keyword only works under the concept of inheritance.

IQ: why to use super keyword?
-> to differ between instance variable of parent class and instance variable of child class, when the variable names are same, we use super keyword.



exampl10:
class demo {    
    int x=10;      // instance variable of parent classs having same name
}
class sample extends demo    
{   int x=100;            // instance variable  of child class  having same name
     void m1(int x){     //local variable also having  same name
   
     System.out.println("Inside m1 method" );
     System.out.println(x );
     System.out.println(this.x );  //this keyword->// differ ben local variable and instance variable within the same class, HAVING SAME variables name.
     System.out.println(super.x );  //super keyword//it will differ betn parent instance variable and child instance variable, when instance variables hAVE SAME NAME.
      

}
}

OBJECT CLASS:
-if a class in java, doesn't extends any other class, then by default it will extends object class.
-There is no similarity or connection between object class and object creation.

IQ: Where exactly the object class present?
->  The object class is present in a package called as java.lang  , A package according to java is a folder structure.

Note:

-OBJECT CLASS, STRING CLASS, SYSTEM CLASS ARE ALL PART OF java.lang package.


TQ: Which is the default package associated with every java program?
->  java.lang

IMP NOTE:(example9)
-whenever we use a class for object creation WITHIN A java program during compilation, the compiler  will first check the object class, within the same program , if not found it will go to the java.lang package, which is also called as default package, if it is found well and good, if not then compiler throw error, cant find the symbol.
