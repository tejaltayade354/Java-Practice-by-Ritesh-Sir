2) NESTED FOR LOOP
-A for loop within another for loop is called as  nested for loop.

   

class   For8
{
	public static void main(String[] args) 
	{

		for(int i=0;i<=2;i++)          

                {  
                   for(int j=0;j<=1;j++)  
                    {
                      System.out.println(" *");
                }
                   System.out.println( );

        }
}

3) WHILE LOOP:
- if the while condition is true, then the statements within the while wil get executed.If the condition is false, then the statements within the while will not get executed.

SYNTAX:
 while(condition){
//statements;
}


ques: write a java prog to display nos from 1 to 5 using while loop.

class  while2
{
	public static void main(String[] args) 
	{
		int i=1;
		while(i<=5){
			System.out.println("The value of i is" +i);
			 i++;
		}
		
	}
}
