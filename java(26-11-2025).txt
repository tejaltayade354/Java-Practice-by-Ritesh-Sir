import java.util.Collections;

    public static void main(String[] args){
        ArrayList<String> al=new ArrayList<String>();
        al.add("TEJAL");    al.add("UTTAMRAO");      al.add("TAYADE");

-ArrayList 24:  + Swiggy :

-Using comparable at  any point of time, we can sort with respect to 1 obj.
-Whenever we use comparable the sorting logics and business logics are maintained commonly, which is not considered as good coding practice.
-To overcome above challenges, we will prefer to go for comparator.
